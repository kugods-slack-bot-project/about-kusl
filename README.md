# KUSL 🤖

## Description
**KU**GODS **SL**ACK Bot; 고려대학교 GDSC 공식 슬랙 봇
> 2022.09. - 

  <br>

### Summary

* 회의 시작 전 리마인드 기능 (정기 회의 알림/공지)
* 슬랙 채널에 익명 질문/답변 기능 (운영, 개발, 디자인 질문)
* 자주하는 질문(FAQ) 답변 기능

  <br>
